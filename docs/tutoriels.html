Les tutoriels interactifs reposent sur le package R learnr qui transforme un document Quarto ou R Markdown en application Shiny exécutable dans le navigateur. Ils combinent texte, blocs de code que l’étudiant·e peut exécuter, ainsi que des questions à choix multiple.

Un tutoriel learnr contient généralement :

  • des exercices de code à modifier en ligne ;
  • des quiz ou zones de réponse ;
  • des messages de rétroaction automatiques.

Ainsi, aucune installation n’est nécessaire pour les apprenant·es et les enseignant·es peuvent proposer des activités interactives clés en main.

Pour en savoir plus, consultez la documentation de learnr et le guide Quarto Interactive Tutorials.
